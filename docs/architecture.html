Data Flow

1

User Action

Select filter or navigate to a view via router

2

Filter State

useFilters() composable updates shared reactive refs

3

API Request

api.js builds URLSearchParams, axios sends GET request

4

Server Filter

FastAPI applies warehouse, category, status, month filters

5

Validation

Pydantic validates and serializes response models

6

UI Update

Vue refs update, computed properties recalculate, DOM re-renders

API Endpoints

Method Path Filters / Params Description
Inventory
GET /api/inventory warehouse, category List all inventory items
GET /api/inventory/{item_id} - Single item details
Orders
GET /api/orders warehouse, category, status, month List all orders
GET /api/orders/{order_id} - Single order details
Dashboard & Reports
GET /api/dashboard/summary warehouse, category, status, month KPI summary metrics
GET /api/reports/quarterly - Quarterly performance
GET /api/reports/monthly-trends - Month-over-month trends
Demand & Backlog
GET /api/demand - Demand forecasts
GET /api/backlog - Backlog items
POST /api/purchase-orders body Create purchase order
Spending & Financial
GET /api/spending/summary - Total spending stats
GET /api/spending/monthly - Monthly spending breakdown
GET /api/spending/categories - Spending by category
GET /api/spending/transactions - Recent transactions
